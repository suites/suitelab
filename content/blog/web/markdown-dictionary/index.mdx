### 제목 Headers

`#`와 숫자를 사용하여 제목을 작성할 수 있다.

{

<table>
  <thead>
    <tr>

    </tr>
  </thead>
  <tbody>
    <tr>
      <td># H1</td>
      <td>
        <h1>H1</h1>
      </td>
    </tr>
    <tr>
      <td>## H2</td>
      <td>
        <h2>H2</h2>
      </td>
    </tr>
    <tr>
      <td>### H3</td>
      <td>
        <h3>H3</h3>
      </td>
    </tr>
    <tr>
      <td>#### H4</td>
      <td>
        <h4>H1</h4>
      </td>
    </tr>
    <tr>
      <td>##### H5</td>
      <td>
        <h5>H5</h5>
      </td>
    </tr>
    <tr>
      <td>###### H6</td>
      <td>
        <h6>H6</h6>
      </td>
    </tr>
  </tbody>
</table>
}

### 인용 Blockquotes

`>`을 사용하여 인용 문구를 작성할 수 있다.

{

<table>
  <thead>
    <tr>
      <th>마크다운</th>
      <th>실행결과</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>&gt; 인용문</td>
      <td>
        <blockquote> 인용문 </blockquote>
      </td>
    </tr>
    <tr>
      <td>&gt;&gt; 인용문안의 인용문</td>
      <td>
        <blockquote>
          {' '}
          인용문1 <blockquote> 인용문2 </blockquote>
        </blockquote>
      </td>
    </tr>
  </tbody>
</table>
}

### 강조 Emphasis

기울여 쓰기(italic) : `*` 또는 `_`로 감싼 텍스트
굴게쓰기(bold) : `**` 또는 `__`로 감싼 텍스트

{

<table>
  <thead>
    <tr>
      <th>마크다운</th>
      <th>실행결과</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>
        *기울여쓰기(italic)*
        <br />
        _기울여쓰기(italic)_
        <br />
        <br />
        **굵게쓰기(bold)**
        <br />
        __굵게쓰기(bold)__
      </td>
      <td>
        <em>기울여쓰기(italic)</em>
        <br />
        <em>기울여쓰기(italic)</em>
        <br />
        <br />
        <strong>굵게쓰기(bold)</strong>
        <br />
        <strong>굵게쓰기(bold)</strong>
      </td>
    </tr>
  </tbody>
</table>
}

### 수평선 Horizontal Rules

`-` 또는 `*` 또는 `_` 을 3개 이상 작성
(단, `-`을 사용할 경우 header로 인식할 수 있으니 이 전 라인은 비워두어야한다.)

{

<table>
  <thead>
    <tr>
      <th>마크다운</th>
      <th>실행결과</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>---</td>
      <td>
        <hr />
      </td>
    </tr>
    <tr>
      <td>***</td>
      <td>
        <hr />
      </td>
    </tr>
    <tr>
      <td>___</td>
      <td>
        <hr />
      </td>
    </tr>
  </tbody>
</table>
}

### 리스트 Lists

#### 순서 있는 리스트 Ordered Lists

`No. ` 숫자 다음 .을 찍는다. (적힌 숫자랑 상관없이 순서대로 번호가 매겨진다.)

{

<table>
  <thead>
    <tr>
      <th>마크다운</th>
      <th>실행결과</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>
        1. list item 1<br />
        2. list item 2<br />
        3. list item 3<br />
        4. list item 4<br />
        5. list item 5<br />
      </td>
      <td>
        <ol>
          <li>list item 1</li>
          <li>list item 2</li>
          <li>list item 3</li>
          <li>list item 4</li>
          <li>list item 5</li>
        </ol>
      </td>
    </tr>
  </tbody>
</table>
}

`*`, `+`, `-` 으로 시작
